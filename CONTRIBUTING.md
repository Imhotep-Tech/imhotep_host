# Contributing to Imhotep Host

First off, thank you for considering contributing to Imhotep Host! It's people like you that make open-source such a great community.

### Where do I go from here?
Currently, the project is in its initial architectural phase. 
Once the V1 core engine is stabilized, we will open up issues for:
- Building new Community Dockerfile Templates (Django, Flask, React, .NET, etc.).
- Expanding the React UI dashboard.
- Integrating Tailscale sidecars alongside Cloudflare.

If you find a bug or have a feature request, please open an Issue using the GitHub tracker. Pull Requests will be warmly welcomed once the core engine is fully committed!
